# Puma X Hyrox Registration Website

## Overview
Full-stack MERN application for Puma X Hyrox event registration following the specified user journey.

## Tech Stack
- Frontend: React.js (Create React App)
- Backend: Node.js + Express
- Database: MongoDB
- Colors: 
  - Teal: #65D2CA
  - Green: #AAC85C
  - Yellow: #F2CB40

## Setup
1. Clone or navigate to puma-hyrox-app
2. Backend: cd server && npm install
3. Frontend: cd client && npm install
4. Configure .env
5. Run backend and frontend.

## Hosting
- Frontend: Vercel/Netlify (free)
- Backend: Render.com or Railway (free tier)
- Domain: Free subdomain from Render/Vercel or freenom/000webhost.
